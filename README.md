პროექტი: ორი ერთნაირი ჟანრის ფილმის შედარება რეიტინგების მიხედვით

ამ პროექტის მიზანი იყო ორი ერთნაირი ჟანრის ფილმის შედარება მონაცემებზე დაყრდნობით. 
კვლევისთვის გამოვიყენე ფილმების რეიტინგები Rotten Tomatoes-დან და IMDb-დან, რათა დამენახა, 
როგორ განსხვავდება კრიტიკოსებისა და მაყურებლის შეფასებები. პროექტი მოიცავს სამ ძირითად ეტაპს: მონაცემების შეგროვებას, დამუშავებასა და ვიზუალიზაციას.

კვლევის საგნად ავირჩიე ორი სამეცნიერო ფანტასტიკის ჟანრის ფილმი: ,,Inception'' და ,,Interstellar.'' 
ეს ფილმები ერთიდაიგივე ჟანრს ეკუთვნის, თუმცა ხშირად განსხვავებულად ფასდება, რაც მათ საინტერესო შესადარებელ ობიექტებად აქცევს.

რატომ ავირჩიე ეს თემა?

ეს თემა ავირჩიე, რადგან:
  ფილმები ძალიან პოპულარულია და განხილვის საგანია საზოგადოებაში.
  Rotten Tomatoes და IMDb წარმოადგენს სანდო წყაროებს ფილმების შეფასებისათვის.
  კრიტიკოსებისა და მაყურებლის შეფასებებს შორის განსხვავება ხშირად საინტერესო ტენდენციებს აჩვენებს.
  პროექტი საშუალებას მაძლევდა პრაქტიკაში გამომეყენებინა მონაცემთა შეგროვებისა და ანალიზის უნარები.

პირველი ეტაპი-მონაცემების შეგროვება:
მონაცემების შეგროვების ეტაპზე გამოვიყენე OMDb API, 
რომელიც იძლევა ინფორმაციას ფილმების შესახებ, მათ შორის Rotten Tomatoes-ისა და IMDb-ის რეიტინგებს.

გამოყენებული ბიბლიოთეკები:
  requests-API-სთან კომუნიკაციისთვის
  pandas-მონაცემების სტრუქტურირებისა და შენახვისთვის

ამ ეტაპზე:
  გავაგზავნე მოთხოვნა OMDb API-ზე.
  ამოვიღე შემდეგი მონაცემები: ფილმის დასახელება, წელი, ჟანრი, Rotten Tomatoes-ის რეიტინგი და IMDb რეიტინგი;
  მონაცემები შევინახე CSV ფაილში სახელწოდებით two_movies_comparison.csv.

მეორე ეტაპი-მონაცემების დამუშავება:
  მეორე ეტაპზე შემოვიტანე CSV ფაილი Jupyter Notebook-ში და დავამუშავე მონაცემები pandas ბიბლიოთეკის გამოყენებით.

სამი ძირითადი მიგნება:
  IMDb-ის რეიტინგის გადაყვანა 0–100 სკალაზე, რადგან IMDb შეფასება 0-დან 10-მდეა, 
  ხოლო Rotten Tomatoes 0-დან 100-მდე, გავამრავლე IMDb რეიტინგი 10-ზე, რათა ორივე შეფასება ერთ სკალაზე ყოფილიყო.
  განსხვავება Rotten Tomatoes-სა და IMDb-ს შორის-გამოვთვალე აბსოლუტური განსხვავება ორ რეიტინგს შორის თითოეული ფილმისთვის, 
  რათა დამენახა, რამდენად განსხვავდება კრიტიკოსებისა და მაყურებლის შეფასებები.
  რომელი ფილმი შეფასდა უფრო მაღლა-დავადგინე, რომელ ფილმს ჰქონდა უფრო მაღალი Rotten Tomatoes-ის რეიტინგი.
  
მესამე ეტაპი-ვიზუალიზაცია:

ვიზუალიზაციისთვის გამოვიყენე matplotlib ბიბლიოთეკა.

შევქმენი სამი ვიზუალიზაცია:
  ბარ-ჩარტი Rotten Tomatoes vs IMDb რეიტინგების შესადარებლად.
  ბარ-ჩარტი რეიტინგებს შორის განსხვავების საჩვენებლად.
  შედარებითი გრაფიკი, სადაც ჩანს, რომელი ფილმი ჯობნის მეორეს საერთო შეფასებით.

დასკვნა
  ამ პროექტის საშუალებით ვისწავლე:
  API-დან მონაცემების შეგროვება.
  მონაცემების დამუშავება pandas-ით.
  ვიზუალიზაციების შექმნა matplotlib-ით.
  მონაცემებზე დაფუძნებული ფილმების ობიექტური შედარება.

პროექტმა აჩვენა, რომ მონაცემთა ანალიზი შეიძლება ეფექტურად გამოვიყენოთ კულტურული თემების, მაგალითად ფილმების, შესასწავლად.
